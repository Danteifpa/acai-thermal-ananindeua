# AçaíThermal - Plataforma de Simulação Científica

Este é um projeto acadêmico desenvolvido para o curso de **Bacharelado em Ciência e Tecnologia (BCT)** do **IFPA Campus Ananindeua**.

## Objetivo
O AçaíThermal é uma ferramenta voltada para a validação térmica do processo de branqueamento da polpa de açaí. Utilizando modelos matemáticos baseados na Lei do Resfriamento de Newton, o sistema auxilia na garantia da segurança alimentar, visando a inativação do *Trypanosoma cruzi* e outros patógenos através do monitoramento rigoroso da temperatura crítica de 52.5°C.

## Funcionalidades
- **Laboratório Virtual:** Simulação em tempo real da curva de resfriamento.
- **Gestão de Campo:** Mapeamento e registro de batedouros em Ananindeua/PA.
- **Academia de Treinamento:** Desafios interativos para capacitação em segurança térmica.
- **Protocolos de Segurança:** Guia completo de boas práticas e normas técnicas.

## Tecnologias Utilizadas
- **Frontend:** React, TypeScript, Tailwind CSS, Shadcn/UI.
- **Backend/Database:** Supabase.
- **Iconografia:** Lucide React.
